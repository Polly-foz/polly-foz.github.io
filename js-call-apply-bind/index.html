JavaScript中call、apply、bind 的用法

Written by Polly-foz with ♥ on January 23, 2020 in JavaScript

1. 用法

Function.prototype.call(),Function.prototype.apply(),Function.prototype.bind()是所有函数共有的方法。

这三个函数存在的意义是改变函数执行时的上下文。简单说，这三个函数可以改变函数运行时this的指向。

call apply bind
语法 function.call(thisArg, arg1, arg2, …) func.apply(thisArg, [argsArray]) function.bind(thisArg[, arg1[, arg2[, …]]])
作用 使用一个指定的 this 值和单独给出的一个或多个参数来调用一个函数。 使用一个指定的 this 值和作为一个数组（或类似数组对象）提供的参数来调用一个函数。 创建一个绑定this值和初始参数值的新函数

2. 区别

  1. call() 与 apply() 方法类似，只有一个区别，就是 call() 方法接受的是一个参数列表，而 apply() 方法接受的是一个包含多个参数的数组。
  2. bind()方法与另外两个方法的区别是:
    1. bind改变了函数执行上下文后没有立即执行函数
    2. bind绑定了this和参数值后，无论怎么调用，绑定的this和参数都不会被改变。
    3. bind绑定的参数值会从arguments[0]开始存储。调用绑定函数时，显式传入的参数会从arguments中存储的绑定参数后面开始存储。
    4. bind创建了一个新函数

3. 示例

3.1. 用call实现继承

function Animal(name){
    this.name = name
    this.run = function(){console.log(`${this.name} is running...`)}
}

function Cat(name){
    Animal.call(this, name)
    this.category = 'Cat'
    this.speak = function(){console.log('miao~')}
}

function Dog(name){
    Animal.call(this, name)
    this.category = 'Dog'
    this.speak = function(){console.log('wang~')}
}

let catty = new Cat('catty')
let doggy = new Dog('doggy')
console.log(catty.name)//catty
console.log(doggy.name)//doggy
catty.run()//catty is running...
doggy.run()//doggy is running...

3.2. 用call和apply方法传递参数，并且指定上下文的this

function A(name){
    this.name = name
    this.showInfo = function(){
        console.log("name: ",this.name)
        console.log("age: ",arguments[0])
        console.log("gender: ",arguments[1])
    }
}

let a = new A("a")
let b = {
    name: 'b'
}

a.showInfo(3,'female')//name:a,age:3,gender:female
a.showInfo.call(b,18,'male')//name:b,age:18,gender:male
a.showInfo.apply({name:'c'},[9,'female'])//name:c,age:9,gender:female

3.3. 用bind方法给函数绑定this和参数

对象中的方法被拿出来后，再调用时，方法中的this不再是原来的对象。用bind方法可以将原来的this绑定在拿出来的方法上。

this.name = 'window' // 浏览器中，this指向全局的window对象
let a = {
    name: 'a',
    getInfo: function(age){
        console.log(`name:${this.name}, age:${age}`，gender:arguments[1])
    }
}

a.getInfo()//name:a, age:undefined, gender:undefined

let retrieveInfo = a.getInfo
retrieveInfo(3)//name:window, age:3, gender:undefined
//函数被取出后，this的值变为全局的window

retrieveInfo.bind(a,10)
retrieveInfo()//name:window, age:undefined, gender:undefined
//bind不改变原函数，而是创建了一个新的函数

let bindedRetrieveInfo = retrieveInfo.bind(a,10)
bindedRetrieveInfo('male')//name:a, age:10, gender:male
//bind给bindedRetrieveInfo绑定了this为a,绑定了形参age（即arguments[0]）的值为10，传入的'male'可以在函数内通过arguments[1]获得
