Js Event

Written by Polly-foz with ♥ on March 1, 2020 updateAt Mar 1, 2020 in JavaScript

需求

点击div1内部（包括div2,div3），则div1变为selected状态（背景变为绿色）。

html和css部分

<div class="div1">
  1
   <div class="div2">
    2
        <div class="div3">
            3
        </div>
    </div>
</div>

<div class="div1">
  1
   <div class="div2">
    2
        <div class="div3">
            3
        </div>
    </div>
</div>

.div1{
  border:1px solid black;
  width:150px;
  height:150px;
}

.div2{
  border: 1px solid black;
  width:100px;
  height:100px;
  margin:0 auto;
}

.div3{
  border: 1px solid black;
  width:50px;
  height:50px;
  margin:0 auto;
}

.selected{
  background-color: green;
}

错误的js代码

document.querySelectorAll(".div1").forEach(div1=>{
  div1.addEventListener("click",function(e){
    const arr = e.target.className.split(' ')
    const index = arr.indexOf('selected')
    if(index < 0){
      arr.push('selected')
    }else{
      arr.splice(index,1)
    }
    e.target.className = arr.join(' ')
  })
})
  • target获取到的是被点击的dom元素
  • currentTarget获取到的是绑定监听器的dom元素

那么直接在currentTarget上修改class不就行了吗？ 悲哀的是，不同于target，currentTarget是只读的。

….好吧，修改currentTarget的属性是可行的。。。
