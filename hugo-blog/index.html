Hugo搭建个人博客

Written by Polly-foz with ♥ on November 6, 2019 in 无

Hugo官方教程

1. 安装Hugo

不同操作系统上的安装教程
以Windows为例

1.1. 下载Hugo release

  • 进入Hugo release页面
  • 下载与系统对应版本
  • 解压，把hugo.exe放到…\hugo目录下
    • …为任意本地路径，建议全英文，不要有空格
    • 以D:\Software\hugo\hugo.exe为例

1.2. 配置环境变量

右击开始 > 系统 > 高级系统设置 > 环境变量 > 系统变量 > 选中Path > 编辑 > 新建 > 填入 D:\Software\hugo (你的hugo.exe所在路径)

1.3. 检查hugo是否成功安装

重启终端，运行hugo version查看版本

2. 创建站点

hugo new site blog

在终端进入你想要存放本地站点的路径，运行上述指令，在当前路径创建blog目录，存放本地站点。

3. 添加主题

可以在hugo官网上挑选心仪的主题， 以LeaveIt为例

  • 下载主题
    进入…/blog目录，输入以下指令

    cd themes
git clone https://github.com/liuzc/LeaveIt.git

  • 设置主题
    打开blog目录下的config.toml文件，根据官方示例配置你的config.toml

4. 发布第一篇文章

  • 进入…/blog目录，输入以下指令
    hugo new posts/my-first-post.md

  • 进入…/blog/content/posts目录，找到my-first-post.md，为my-first-post.md添加一些内容，如：

    ---
title: "我的第一篇博客"
date: 2019-11-06T17:33:30+08:00
draft: true
---
# 第一个标题
* 第一个列表
* 1
* 2
* 3

5. 启动Hugo Server

  • 在终端进入…/blog目录，输入以下指令
    hugo server -D -D表示允许查看草稿
  • 启动成功后，在浏览器访问http://localhost:1313/，就可以查看生成的本地站点

6. 生成静态页面

  • 将my-first-post.md开头的draft值改为false，表示要将my-first-post.md发布到博客中给其他人看，而不是存为草稿
  • 在终端进入…/blog目录，输入以下指令
    hugo
  • 生成的静态页面会保存在当前目录的public目录下

7. 将静态站点配置到Github Pages

  • 在GitHub上新建一个名为xxx.github.io的仓库（xxx为你的GitHub用户名）
  • 将../blog/public文件夹作为项目提交到仓库
  • 进入xxx.github.io仓库的Settings，设置源为master分支
  • 通过网址xxx.github.io即可访问你的博客
