Vue .sync修饰符

Written by Polly-foz with ♥ on February 13, 2020 in Vue

.sync修饰符是vue2.3.0简化的一个语法糖。

用法举例：

  • 点击hide隐藏方框及其内容
  • 点击toggle切换方框及其内容显示or隐藏

main.js

import Vue from "vue";
import Son from "./components/Son.vue";

Vue.config.productionTip = false;

new Vue({
  data: {
    visible: true
  },
  components: { Son },
  template: `
  <div id="app">
    <Son :visible='visible' @update:visible="value => visible = value"></Son>
    <button @click="visible=!visible">toggle</button>
  </div>
  `
}).$mount("#app");

Son.vue

<template>
  <div id="son" v-if="visible">
    <p>son~</p>
    <button @click="$emit('update:visible',false)">hide</button>
  </div>
</template>

<script>
export default {
  name: "Son",
  props: ["visible"],
  watch: {
    hide() {
      console.log("hide change " + this.visible);
    }
  }
};
</script>
<style scoped>
#son {
  width: 10rem;
  height: 10rem;
  border: 1px solid black;
  padding: 1rem;
}
</style>

使用.sync修饰符，则main.js中

<Son :visible='visible' @update:visible="value => visible = value"></Son>

可简化为

<Son :visible.sync='visible'></Son>
