for(let i = 0; i < 10; i++){
    setTimeout(()=>console.log("let: ",i),1000)
}
for(var j = 0; j < 10; j++){
    setTimeout(()=>console.log("var: ",j),1000)
}

输出：

从上面的例子可以看见声明关键字的不同，带来了截然不同的结果。

ES6新增了let命令，let用法类似于var。let与var不同的是，let声明一个作用域被限制在块级中的变量、语句或者表达式，而var明的变量只能是全局或者整个函数块的。var不支持块级作用域。

在ES6之前，可以利用闭包的特性来模拟块级作用域。

for(var i = 0; i < 10; i++){
    (function(i){
        setTimeout(()=>console.log(i),1000)
    })(i)
}
