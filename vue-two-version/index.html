vue.js 与 vue.runtime.js

Written by Polly-foz with ♥ on February 7, 2020 in

1. 区别

方方老师的总结：

Vue 完整版 Vue 非完整版 评价
特点 有compiler 没有compiler compiler占40%体积
视图 写在HTML里，或者写在template选项 写在render函数里，用h来创建标签 h是尤雨溪写好传给render的
cdn引入 vue.js vue.runtime.js 文件名不同，生成环境后缀名为.min.js
webpack引入 需要配置alias 默认使用此版 尤雨溪配置的
@vue/cli引入 需要额外配置 默认使用此版 尤雨溪、蒋豪群配置的

最佳实践：

使用vue.runtime.js + compiler，compiler会将template编译成render

2. template和render

2.1. template使用

使用 vue.js 或者 vue.runtime.js + compiler 时，可以用template。

public/index.html body部分

<body>
    <noscript>
      <strong>We're sorry but <%= htmlWebpackPlugin.options.title %> doesn't work properly without JavaScript enabled. Please enable it to continue.</strong>
    </noscript>
    <div id="demo-2version">
      <div>这是demo-2version内的html内容</div>
    </div>
    <!-- built files will be auto injected -->
</body>

scr/main.js

import Vue from 'vue'

Vue.config.productionTip = false

new Vue({
    el:'#demo-2version',
    template:`
        这是template给出的模版-div外
        <img src="" alt="hi">
        <a href="">template-a</a>
        <div>这是template给出的模版-div1</div>
        <div>这是template给出的模版-div2</div>
        这是template给出的模版-div外
    `
})

运行后生成的html body部分

<body>
    <noscript>
      <strong>We're sorry but vue-demo-1 doesn't work properly without JavaScript enabled. Please enable it to continue.</strong>
    </noscript>
    <img src="" alt="hi">
    <!-- built files will be auto injected -->
  <script type="text/javascript" src="/js/chunk-vendors.js"></script><script type="text/javascript" src="/js/app.js"></script>

</body>

NOTE：

template里的第一个标签替换了挂载元素

2.2. render使用

使用vue.runtime.js 或 vue.runtime.js + compiler，可以用render。

public/index.html内容同2.1.

新增文件src/Demo-2version.vue

<template>
    <div>
        这是template给出的模版
        <img src="" alt="hi">
        <a href="">template-a</a>
        <div>这是template给出的模版-div1</div>
        <div>这是template给出的模版-div2</div>
        这是template给出的模版
    </div>
</template>

<script>
    export default {
        name: "demo-2version"
    }
</script>

<style scoped>

</style>

scr/main.js

import Vue from 'vue'
import Demo2Vesion from './Demo-2version.vue'

Vue.config.productionTip = false

new Vue({
    el:'#demo-2version',
    render:h=>h(Demo2Vesion)
})

运行后生成的html body部分

<body>
    <noscript>
      <strong>We're sorry but vue-demo-1 doesn't work properly without JavaScript enabled. Please enable it to continue.</strong>
    </noscript>
    <div data-v-a93c0c32=""> 这是template给出的模版 <img data-v-a93c0c32="" src="" alt="hi"><a data-v-a93c0c32="" href="">template-a</a><div data-v-a93c0c32="">这是template给出的模版-div1</div><div data-v-a93c0c32="">这是template给出的模版-div2</div> 这是template给出的模版 </div>
    <!-- built files will be auto injected -->
  <script type="text/javascript" src="/js/chunk-vendors.js"></script><script type="text/javascript" src="/js/app.js"></script>

</body>

NOTE：

src/Demo-2version.vue中的内容必须有个根标签，用于替换挂载元素

优先级

优先级：render > template > 挂载元素的内容
